
# GMAO Web Moderne – Site Interne (Gratuit)

## Stack
- Backend : Flask (API REST)
- Frontend : React + Vite
- Base de données : SQLite (compatible PostgreSQL)
- Hébergement gratuit : Render

## Lancement local
### Backend
```bash
cd backend
pip install flask flask_sqlalchemy flask_cors
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
